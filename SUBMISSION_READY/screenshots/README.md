# Screenshots Directory

## Required Screenshots

Please add the following screenshots to the appropriate folders:

### 1. ER Diagram
- **File:** `er_diagram.png`
- **Content:** ER diagram showing all 7 tables with relationships
- **Requirements:** Must show PK, FK, and cardinalities

### 2. Business Process Diagram
- **File:** `business_process.png`
- **Content:** BPMN diagram with swimlanes
- **Requirements:** Must show workflow from start to end

### 3. Database Structure
- **Folder:** `database_structure/`
- **Files:**
  - `database_tables.png` - All tables in SQL Developer
  - `table_structure.png` - Table structure details

### 4. Sample Data
- **Folder:** `sample_data/`
- **Files:**
  - `customers_data.png` - 5-10 rows from customers table
  - `readings_data.png` - 5-10 rows from meter_readings table
  - `bills_data.png` - 5-10 rows from bills table
  - `payments_data.png` - 5-10 rows from payments table

### 5. Procedures and Triggers
- **Folder:** `procedures_triggers/`
- **Files:**
  - `procedure_code.png` - Show procedure code in editor
  - `trigger_code.png` - Show trigger code in editor
  - `package_code.png` - Show package code in editor

### 6. Test Results
- **Folder:** `test_results/`
- **Files:**
  - `test_execution.png` - Test script execution results
  - `function_test.png` - Function test results
  - `procedure_test.png` - Procedure test results

### 7. Audit Logs
- **Folder:** `audit_logs/`
- **Files:**
  - `audit_log_entries.png` - Audit log table contents
  - `denied_operations.png` - Denied operations
  - `allowed_operations.png` - Allowed operations

## ⚠️ IMPORTANT

**All screenshots must show:**
- Project name or database name
- Clear, readable text
- Complete information
- Professional appearance

## How to Take Screenshots

1. **Windows:** Use Snipping Tool or Win+Shift+S
2. **Mac:** Use Cmd+Shift+4
3. **SQL Developer:** Use built-in screenshot feature

## File Naming

Use descriptive names:
- `er_diagram.png` (not `screenshot1.png`)
- `customers_data.png` (not `img001.png`)
